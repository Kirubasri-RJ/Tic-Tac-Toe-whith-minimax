# Tic-Tac-Toe with Minimax Algorithm
## Codmetric AI Internship - Task 2

## Description
A Tic Tac Toe game where the computer makes 
optimal moves using the Minimax algorithm.

## Features
- Console based game board
- Minimax algorithm with recursion
- Player vs Computer
- Displays game state after each move
- Shows winner or draw at the end

## Tools Used
- Python
- Google Colab

## How to Play
1. Run the code in Google Colab
2. You are X, Computer is O
3. Enter position number 1-9
4. Computer plays automatically
